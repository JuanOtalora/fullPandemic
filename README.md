# Pandemic

A simple idle game based on Cookie Clicker and games of the sort, It has a backEnd done with Express and the front is made with React


#Running

To run you must first do:
  cd back_pandemic
  npm install
  npm start
  
To run the front end:
  cd front
  npm start
